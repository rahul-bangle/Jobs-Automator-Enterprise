# 🏢 PROJECT.md — Jobs Automator Enterprise

## What This Is
A fully autonomous, AI-driven job discovery and application pipeline.

## Core Value
Transforming a simple search query into hundreds of high-quality, ATS-optimized job applications with zero manual intervention.

## 🔧 Technical Stack
- **Backend**: Python 3.11 / FastAPI
- **Intelligence**: Groq (70B/8B), Crawl4AI
- **Storage**: SQLite (Local), Supabase (Remote)
- **Frontend**: React / Vite (Virtualized Result Table)

## 📋 Requirements
- [x] Multi-site job discovery (JobSpy)
- [x] Semantic job description parsing (Crawl4AI)
- [x] White-box job scoring (Groq)
- [x] Recursive self-healing optimization loop
- [x] Professional PDF generation (Reactive Resume Schema)
- [/] Autonomous submission agent (BrowserUse/Playwright)
- [/] High-performance Result Table (1,000+ jobs)

## 🔓 Constraints
- Must maintain ATS score > 75% for all applications.
- Safety-gated automated submissions (Human-in-the-loop fallback).
- Local-first architecture with optional cloud sync.
